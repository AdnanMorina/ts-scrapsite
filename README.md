# Birthday Scrapbook

A collaborative digital scrapbook built a beautiful girl's birthday. Each contributor gets their own page with photos, a Spotify track, and a personal message.

---

## Tech

- React 18 via CDN (no build step, no Node required to contribute)
- Babel Standalone for JSX
- Pure CSS with custom properties — no Tailwind, no frameworks
- Deployed on Vercel via this repo
